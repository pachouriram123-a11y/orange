## 📌 Project Overview

This project predicts **loan default risk** using **Logistic Regression (Ridge Regularization)** in Orange Data Mining.

The objective is to classify customers into:

- **0 → Non Default**
- **1 → Default**

---

## 🛠 Tech Stack

- Orange Data Mining
- Logistic Regression
- CSV Dataset
- ROC Analysis
- Confusion Matrix
